# Red Hat Decision Central

As known as the Workbench.

## Quick Start

Install JBoss EAP and RHDM Central:
```
./install.sh
```

Apply patches:
```
./patch.sh
```

Add 'admin' and 'controller' users:
```
./users.sh
```

Review the setup.properties file and run:
```
./setup.sh
```

Start the server:
```
./run.sh
```

## More options

Install offline, without downloading files:
```
./install.sh offline
```
